# Security

## サポート対象

最新の `main` ブランチを前提とします。古いコミットへのバックポートは保証しません。

## 報告方法

脆弱性の可能性がある内容は、GitHub の **Security advisories** から報告するか、リポジトリの Issues で「公開して問題ない範囲」から共有してください。プライベートで連絡したい場合は、Maintainer の GitHub プロフィールから連絡手段を確認してください。

## 想定する脅威モデル

このツールは **ローカル実行**（Ollama + SQLite）を前提にしています。外部クラウド API に服画像を送らない設計ですが、パソコン上のマルウェアや共有マシンでの画面盗み見などは、一般的な OS の対策の範囲になります。
