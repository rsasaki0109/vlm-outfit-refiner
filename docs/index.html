Local outfit recommender

手持ちの服の写真をローカル VLM で属性化し、シチュエーションに合わせたコーデ案を 無難 / きれいめ / 攻めの 3 通り、理由付き（JSON）で返すツール。 外部クラウド API は使いません（Ollama 前提）。

Python 3.10+
Ollama Vision
SQLite local DB
JSON I/O
CLI + Streamlit
Quickstart UI Commands
README banner

1) 服画像を登録

ローカル画像 → VLM で属性 JSON → SQLite へ保存。重複は SHA-256 で回避。

python main.py add ./photos/shirt.jpg

2) 3パターン提案

tops×bottoms×shoes の組み合わせをスコアリングし、上位から 3 案＋理由を出力。

python main.py recommend

3) 手直しも簡単

VLM の誤分類は reclassify（再推論）か edit（手修正）でリカバリできます。

python main.py edit 2 --category bottoms
